# JavaScript Instructions

- Follow existing module system (ESM vs CommonJS) and linting style.
- Use existing test framework (Jest/Vitest/Mocha) found in repository.
- For static methods in classes, call `ClassName.method(...)`; avoid instance-based invocation for static behavior.
- Keep tests in existing `test`, `tests`, or `__tests__` conventions.
- Add assertions for normal and edge cases.
